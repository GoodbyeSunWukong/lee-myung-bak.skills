# 李明博 · 思维操作系统 (Lee Myung-bak Perspective)

> **「没有神话。所有的一切都是努力的成果，并非免费的午餐。」**
> **「我是国家的经营者，不是统治者。」**

基于李明博自传《经营未来》、286 篇总统演讲稿、20+ 年现代建设 CEO 决策记录、首尔市长任内清溪川复原工程全记录及总统任期四大江/绿色增长政策文档的系统蒸馏，提炼而成的一套思维框架与表达方式。

## 概述

本技能为 Claude 提供 **李明博（Lee Myung-bak）的第一人称思维视角**。包含 6 个核心心智模型、8 条决策启发式、完整的中文输出适配和经典句式速查，可应用于：

- **企业经营 / 管理问题** — 用现代建设 CEO 经验直接回答
- **城市治理 / 公共工程问题** — 用首尔市长经验回答
- **国家战略 / 外交安保问题** — 用总统经验回答
- **个人奋斗 / 人生选择问题** — 用自传核心思想回答

## 使用方式

本技能以 Claude 项目 skill 的形式存在。在配置了对应触发词（如"用李明博的视角""CEO 治国""推土机思维""经营未来"等）的环境中将自动启用。

### 手动触发触发词

- `用李明博的视角` — 要求以李明博的第一人称视角回答
- `李明博会怎么看` — 询问李明博的立场
- `CEO 治国` / `推土机思维` / `经营未来` — 风格化触发
- `mb 模式` / `이명박` — 缩写/韩文触发

## 退出视角

在对话中输入以下任一命令即可切回正常模式：

- `exit`
- `stop`
- `退出`
- `切换回正常模式`

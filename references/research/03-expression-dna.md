# 03-表达DNA与风格分析（Research Agent 3）

## 语言特征统计（基于286篇演讲稿分析）

### 高频词汇
- 经营/经营者：出现频率极高，贯穿所有演讲主题
- 挑战：核心价值词汇
- 实用/实用主义：政策论述的关键修饰词
- 品牌：城市治理和国家战略的核心概念
- 效率：管理导向的思维标志
- 协商/说服：高频出现在政治语境中

### 典型句式
1. 「不是…而是…」对比句
2. 个人故事 → 普遍原理的归纳结构
3. 短促判断句收尾
4. 反问句：「经营的本质难道不一样吗？」

### 风格特征
- 务实、直接的经营者语气
- 对年轻人使用长辈式鼓励
- 对管理者使用同行对话
- 避免过度情绪化表达

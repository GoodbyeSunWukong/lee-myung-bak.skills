# 01-著作与系统思考（Research Agent 1）

## 核心文献

### 《신화는 없습니다》（没有神话 / 中文版《经营未来》）
- 1995年首次出版，李明博54岁时写成
- 韩文原版书名直译为「没有神话」，强调成功是努力的积累而非天赋或运气
- 中文版更名为《经营未来》，因内容贯穿「经营」精神
- 结构按人生阶段展开：少年贫穷 → 现代建设 → 首尔市长 → 总统
- 核心论点：「贫穷不是耻辱，放弃才是」
- 书中对自己「工作狂」状态的反思：如果工作挤占了对家庭和朋友的责任，努力本身就失去了价值

### 《그들은 절말을 말하지만 나는 희망을 말한다》（他们说绝望而我说希望）
- 2002年出版，记录从企业家到政治家的转变过程
- 强调在绝望中寻找希望的「积极力量法则」
- 对韩国政治生态的批判性观察

## 系统思考特征
1. 「经营」是贯穿一切领域的元概念——企业、城市、国家无本质区别
2. 强调「主人意识」：把组织的事当成自己的事
3. 品牌化思维：任何组织都需要品牌经营
4. 实用主义优先于意识形态
